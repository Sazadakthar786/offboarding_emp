# 🚀 Employee Offboarding Automation System

A smart, modular offboarding automation project built using **Cursor**. This system simplifies the entire employee exit process—connecting HR, IT, Finance, and Admin into one seamless workflow.

---

## 🌟 Overview

Managing employee exits can be complex and time-consuming. This project automates offboarding using a centralized dashboard that integrates with:

- ✅ **n8n** for workflow automation  
- 🧾 **Jira** for exit ticket tracking  
- 🧍 **ZenHR** for HR updates  
- 🛡️ **Azure AD** for account deactivation  
- 📧 **Email (SMTP)** for multi-stage approvals  

All orchestrated from a clean, developer-friendly frontend built in **Cursor**.

---

## ⚙️ Features

- 🔄 Automated offboarding workflows  
- 📬 Email-based approval system (HR → Admin → Finance)  
- 🧠 Status tracking via connected APIs  
- 📊 Dashboard UI for clear role-based navigation  
- 🔐 Secure, permission-driven integration

---

## 🛠️ Tech Stack

- 💻 Frontend: Cursor (HTML/CSS/JS or React)
- ⚙️ Automation: n8n Workflows
- 🔗 APIs: Jira, ZenHR, Azure AD, SMTP
- ☁️ Deployment: GitHub + n8n Cloud
